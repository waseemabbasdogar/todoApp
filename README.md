📝 To-Do List App (JavaScript)

A simple and interactive To-Do List application built using Vanilla JavaScript, focused on strengthening core frontend fundamentals like DOM manipulation, event handling, and local storage.

🚀 Project Overview

This project demonstrates strong foundational knowledge of JavaScript by implementing:

Dynamic DOM manipulation

Event delegation

Persistent data storage using localStorage

Clean and maintainable logic structure

The primary goal was to reinforce core JavaScript concepts by building a practical, real-world mini application without relying on external libraries or frameworks.

✨ Features

➕ Add new tasks

✅ Mark tasks as completed

❌ Delete tasks

💾 Persist tasks after page refresh

🌗 Theme toggle (Light/Dark mode)

🧠 Technical Concepts Demonstrated
1. DOM Manipulation

Selecting elements using getElementById() and querySelector()

Creating elements dynamically with createElement()

Updating the DOM using appendChild()

Toggling CSS classes for UI state changes

2. Event Handling

Using addEventListener()

Implementing event delegation for efficient event management

Handling click events based on target elements

3. Local Storage API

Storing task data in browser localStorage

Persisting application state after reload

Synchronizing UI with stored data

Updating storage on every state change

4. State Management Thinking

Ensuring completed tasks remain completed after refresh

Keeping UI and storage logic consistent

🛠 Tech Stack

HTML5

CSS3

JavaScript (ES6)

Browser LocalStorage API

No frameworks. No libraries. Pure JavaScript.

📂 How It Works

User enters a task.

A new <li> element is dynamically created.

Click events toggle the .completed class.

The entire task list state is saved to localStorage.

On page load, stored tasks are restored automatically.

While simple in appearance, this project demonstrates:

Strong understanding of JavaScript fundamentals

It reflects a solid grasp of core frontend concepts that scale into larger applications and frameworks like React.
